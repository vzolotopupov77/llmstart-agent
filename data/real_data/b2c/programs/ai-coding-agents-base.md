Подготовка ИТ-специалистов к созданию **production-ready агентных систем** — от понимания архитектуры языковых моделей до разработки автономных агентных и мультиагентных систем с RAG, tool calling, MCP-интеграциями и системами безопасности.

## Целевая аудитория

## Описание программы

*С сайта:* Комплексное освоение современных технологий создания ИИ-агентов. От архитектуры языковых моделей до мультиагентных систем с RAG, tool calling, MCP-интеграциями и системами безопасности.

Комплексная программа по освоению современных технологий создания ИИ-агентов. Программа охватывает полный цикл разработки агентских систем: фундаментальные принципы работы LLM, практическое применение AI-driven методологии, создание RAG-систем с advanced техниками, интеграцию с внешними инструментами, оценку качества и обеспечение безопасности, а также проектирование мультиагентных архитектур.

## Программа курса

### Тема 1. Основы LLM и стандартные API для работы моделями

- Архитектура и принципы работы языковых моделей
- Ограничения моделей: параметрическая память, размер контекста
- Методы работы с моделями: промпт-инжиниринг, контекст-инжиниринг, RAG
- Что такое агент: определение, компоненты, архитектура
- OpenAI API как стандарт де-факто
- Работа с различными провайдерами (OpenAI, OpenRouter, DeepSeek, GigaChat)
- Структурированный вывод (Structured Output, JSON Schema)
- Системные промпты и управление поведением ИИ-ассистента
- Управление контекстом и историей диалога
- Обзор моделей и провайдеров, критерии выбора модели под задачу
- Обзор дорожной карты практики курса

**Практическая работа:**  
Выбор модели для задачи, настройка инференса и создание CLI-бота для работы с LLM через OpenAI-совместимые API.

### Тема 2. AI-driven разработка с ИИ-агентами на примере Cursor

- AI-driven методология и её применение для эффективной разработки с использованием ИИ-агентов
- Настройка Cursor: конфигурация, правила, соглашения, рабочий процесс
- Управление контекстом и эффективные техники промптинга
- От идеи к реализации: планирование итерации, соглашения, списки задач, рабочий процесс
- Разработка по всему жизненному циклу (от идеи до развертывания)
- Управление контекстом для ведения диалога

**Практическая работа:**  
AI-driven разработка ИИ-ассистента с LangChain в Cursor.

### Тема 3. Мультимодальные возможности и локальный запуск LLM

- Обработка голосовых сообщений и транскрибация речи
- Анализ изображений
- Задачи извлечения информации из различных форматов
- Prompt engineering для мультимодальных задач
- In-context learning
- Локальный запуск моделей (Ollama/LMStudio)

**Практическая работа:**  
Реализация мультимодальных возможностей в ИИ-ассистенте. Настройка локальной LLM для тестирования.

### Тема 4. RAG с LangChain: от теории к практике

Паттерны и архитектура:
- Обзор RAG-экосистемы: классификация подходов и архитектур
- RAG pipeline: индексация, chunking, векторизация, поиск, генерация
- Стратегии chunking: размер, перекрытие, семантическое разделение
- Векторные базы данных и embedding модели для семантического поиска
- Naive RAG vs Advanced RAG: эволюция подходов

Создание RAG-ассистента:
- Настройка окружения и загрузка документов
- Индексирование: загрузчики документов, разбиение на чанки, создание векторного хранилища
- Retrieval: поиск релевантных фрагментов документов
- Augmented Generation: создание промптов с контекстом и генерация ответов
- Создание RAG-цепочек для автоматизации процесса
- Диалоговый режим и обработка истории сообщений

**Практическая работа:**  
Проектирование архитектуры и разработка RAG-системы с базой знаний.

### Тема 5. Мониторинг и оценка качества RAG-систем

- Observability для RAG-систем: трейсинг, логирование, метрики
- Базовые метрики оценки качества: Context Precision, Context Recall, Faithfulness, Answer Relevance
- LLM-as-a-Judge: использование LLM для оценки ответов
- RAGAS: фреймворк автоматической оценки RAG-систем
- LangSmith/LangFuse: мониторинг, трейсинг и аналитика запросов в production
- Cost control: подсчет токенов и стоимости запросов
- Способы отладки и анализа проблемных запросов

**Практическая работа:**  
Внедрение системы мониторинга и автоматической оценки качества RAG-системы.

### Тема 6. Advanced RAG

- Query Transformation: переформулирование и декомпозиция запросов
- Query Routing: интеллектуальная маршрутизация запросов в зависимости от типа
- Query Construction: построение структурированных запросов
- Hybrid Search: комбинация семантического и keyword поиска
- Advanced Indexing: продвинутые методы индексирования документов
- Self-RAG: системы с самооценкой качества извлеченного контекста
- Agentic RAG: агентный подход с планированием и адаптацией
- Кейсы применения Advanced RAG в production

**Практическая работа:**  
Улучшение RAG-системы с помощью Advanced техник и разбор production кейсов.

### Тема 7. Агенты с LangChain и LangGraph

- Agent vs Workflow: что и когда применяется
- Tool calling: подключение внешних инструментов к агентам
- Память агента (memory): краткосрочная и долговременная память
- Агентные архитектурные паттерны
- LangGraph: графовые workflows для сложных агентов
- Состояния, узлы, ребра: построение агентского workflow
- ReAct паттерн: Reasoning + Acting в агентском контексте
- Создание Agentic RAG-системы с планированием запросов
- Отладка и трейсинг агентских процессов

**Практическая работа:**  
Проектирование архитектуры и разработка автономного ИИ-агента на LangGraph.

### Тема 8. Model Context Protocol (MCP)

- Model Context Protocol (MCP): стандарт для расширения возможностей LLM
- Архитектура MCP: серверы, клиенты, транспорты
- MCP vs традиционный tool use: ключевые различия
- Создание MCP-серверов: ресурсы, инструменты, промпты
- Экосистема MCP: готовые серверы и коннекторы
- Интеграция ИИ-агентов с внешними системами через MCP

**Практическая работа:**  
Разработка MCP-сервера для интеграции агента с внешними системами.

### Тема 9. Безопасность агентных систем

- Ландшафт угроз для LLM и агентных систем
- Типы атак: Adversarial Prompting (Injection, Jailbreaking), Adversarial Examples, Data Poisoning
- Model Extraction и уязвимости Supply Chain
- Уязвимости агентов и RAG-систем
- Guardrails и политики безопасности: контроль входов и выходов
- Защитные механизмы и фреймворки безопасности (LLMGuard, Giskard)
- Human-in-the-loop 


**Практическая работа:**  
Внедрение систем безопасности в автономный агент. Тестирование на устойчивость к атакам.

### Тема 10. Оценка качества агентов

- Зачем специальные метрики для агентов: отличия от RAG-систем
- Четырехуровневая система оценки (Task, Tool, Trajectory, Topic)
- Tracing как основа для метрик агентов
- Multi-turn Evaluation: оценка диалоговых агентов
- Фреймворки оценки: DeepEval, RAGAS, LangChain (agentevals)
- Стандартизированные бенчмарки и тестовые наборы для агентских систем

**Практическая работа:**  
Внедрение комплексной системы оценки качества агентской системы.

### Тема 11. Переход к мультиагентным системам

- Ограничения одноагентных систем: сложность контекста, множество инструментов, разные экспертизы
- Преимущества мультиагентных систем: модульность, масштабируемость, специализация
- Архитектуры мультиагентных систем: Network, Supervisor, Hierarchical, Custom 
- Координация, коммуникация и специализация агентов
- Мультиагентные системы с LangChain/LangGraph
