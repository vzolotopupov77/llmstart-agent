## Описание программы

Продвинутая образовательная программа для ИТ-специалистов по созданию production-ready агентных систем. Программа фокусируется на переходе от прототипа к работающему MVP через AI-driven разработку.
Охватывает формирование спецификации и AI-кодинг, датасет-менеджмент, GraphRAG с векторными и графовыми БД, мультимодальный RAG, продвинутый context engineering, Deep Agents с planning и делегированием, evaluation и red teaming, prompt management, мультиагентные паттерны и масштабирование через A2A и A2UI протоколы.

**Целевая аудитория:** Разработчики, ИТ-специалисты, Tech Leads, AI-инженеры с базовыми знаниями LLM и агентов

### Ключевые особенности программы:

- **От прототипа к MVP** — планомерное развитие через AI-driven проектирование и реализацию
- **Датасет-менеджмент** — data-driven подход с annotation queues
- **GraphRAG** — гибридный поиск с векторными и графовыми БД
- **Deep Context Engineering** — управление контекстом через файловую систему и MCP
- **Deep Agents** — planning, skills, subagents, long-term memory
- **Evaluation и Red teaming** — component-level evaluation через tracing и тестирование на уязвимости
- **Prompt Management** — централизованное управление с MCP и A/B тестами
- **Масштабирование** — мультиагентные паттерны, A2A и A2UI протоколы

## Что будете уметь после курса

### Проектирование и спецификация
- Формировать спецификацию production-ready агентных систем
- Определять компоненты и архитектуру сложных агентных решений
- Настраивать AI-кодинг агенты с документацией, правилами и контекстами

### Датасет-менеджмент и data-driven подход
- Анализировать данные для базы знаний и диалогов
- Подготавливать валидационные датасеты и строить решения от данных
- Управлять датасетами для обучения и контроля качества

### Векторные и графовые БД, GraphRAG
- Работать с векторными базами данных
- Проектировать Knowledge Graphs
- Реализовывать GraphRAG
- Применять гибридный поиск

### Мультимодальный RAG
- Обрабатывать изображения в RAG-системах
- Работать с Vision API
- Интегрировать мультимодальные embedding модели

### Context Engineering и Deep Agents
- Управлять контекстом через State, Store, файловую систему
- Интегрировать внешние источники через MCP
- Применять dynamic context (выбор tools, models, форматов)
- Реализовывать planning и task decomposition
- Использовать skills и subagents для делегирования
- Работать с long-term memory и checkpoint/resume

### Evaluation и Red teaming
- Проводить component-level evaluation через LLM tracing
- Применять метрики для агентов (Task, Tool, Trajectory, Topic)
- Использовать DeepEval и RAGAS
- Тестировать на уязвимости через red teaming
- Проводить adversarial attacks (Prompt Injection, Jailbreaking)

### Prompt Management
- Централизованно управлять промптами
- Версионировать промпты и проводить A/B тесты
- Использовать MCP для доступа к промптам
- Анализировать производительность через link to traces
- Работать с multimodal промптами

### Мультиагентные системы и масштабирование
- Применять паттерны: Subagents, Handoffs, Router, Custom workflows
- Проектировать распределенные агентные системы
- Использовать A2A протокол для взаимодействия независимых агентов
- Применять A2UI для динамической генерации интерфейсов
